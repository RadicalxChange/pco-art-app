# PCO Art App

# Getting Started

```bash
pnpm install
```

#### Environment Variables

```bash
cp .env.example .env
```
Add the API key of the RPC node provider in `.env`

```bash
`NEXT_PUBLIC_ALCHEMY_API_KEY=`
```
or
```bash
`NEXT_PUBLIC_INFURA_API_KEY=`
```

#### Development

```bash
pnpm dev
```

#### Build

```bash
pnpm build
```
